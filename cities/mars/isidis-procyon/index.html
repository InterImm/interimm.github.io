南河城

南河城是位于火星伊希地的一座重要城市

城市介绍

由于历史原因，南河城的主要功能是服务业。位于城市中心的是星际移民中心的巨型建筑。关于南河城的起源可以参考伊希地城市圈。

目前建设情况可以参考下面的3D模型。可以在我们的 GitHub 创建新的 Issues 来提出你的建筑或者地标设想，并且命名。

关于

星际移民中心是星际移民局的核心机构，从事太阳系开发的各项研究，同时也为火星火星基地提供支持。
